# **App Name**: FinanceFlow

## Core Features:

- Authentication: Secure user authentication via JWT upon login.
- Dashboard View: Display a summary of financial activities, like total balance, recent transactions etc.
- Transaction Management: Create, Read, Update, and Delete financial transactions with fields for sub-category, wallet, type, frequency, amount, and date.
- Main Category Management: Create, Read, Update, and Delete main categories with fields for name and color.
- Sub Category Management: Create, Read, Update, and Delete sub-categories, associated with a main category, with fields for name and color.
- Wallet Management: Create, Read, Update, and Delete wallets with fields for name, currency, amount, and type.
- Transfer Management: Create and Delete transfer transactions between wallets.

## Style Guidelines:

- Primary color: Deep sky blue (#3498db), providing a clean and trustworthy feel for financial data.
- Background color: Light gray (#ecf0f1), offering a subtle backdrop that enhances readability and reduces visual strain.
- Accent color: Emerald green (#2ecc71) to highlight key actions and success states, like completed transfers.
- Body and headline font: 'PT Sans' (sans-serif) offers a balance of modernity and readability, suitable for both titles and content.
- Use consistent, minimalist icons to represent categories, wallets, and transaction types, enhancing visual understanding.
- Maintain a clean, grid-based layout for dashboards and transaction lists, promoting ease of navigation and a sense of order.
- Employ subtle transitions and animations to provide feedback and enhance the user experience, such as sliding panels for transaction details.
